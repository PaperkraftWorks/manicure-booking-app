# Standalone HTML Page

This folder contains a standalone HTML version of the manicure booking app that can be deployed to GitHub Pages or any static hosting service.

## Features

- No build step required
- All functionality included
- Uses Tailwind CSS via CDN
- Vanilla JavaScript
- Fully responsive

## Usage

Simply open `index.html` in a browser or deploy it to any static hosting service.

## GitHub Pages

This folder is configured to be deployed automatically via GitHub Actions when you push to the main branch.
